# WatchMe

Desafio 2 do Bootcamp da [Rocketseat](https://rocketseat.com.br/) - Ignite: ReactJS. Consiste em separar a aplicação em componentes. 

## Requisitos:

* [NodeJS](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/)

## Como usar:

Mock API - JSON Server:

```bash 
yarn server
```

Aplicação: 

```bash 
yarn dev
```
